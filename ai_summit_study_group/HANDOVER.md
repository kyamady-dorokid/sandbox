## 1. これは何か

社内エンジニア向けAI学習会（説明20分＋質疑10分）のスライド。
- **対象**：ChatGPT / Claude Code を触ったことはあるが使いこなしてはいない「入り口」層。
- **狙い**：①AI時代こそ要件定義の重要性が増していると腹落ちさせる ②だから当社は仕様駆動開発（SDD）に取り組む、という方向性を共有する。
- **キーメッセージ**：「AIが速いほど、起点の要件定義が効く」。
- **今日やらないこと**：具体的な導入・運用手順（別途「実践回」で扱う前提）。

成果物：**`ai_study_sdd.pptx`（全20枚, 16:9ワイド）**

## 3. 社用PCでの2つの進め方

### A. PowerPointで直接編集する（推奨・最短）
`.pptx` をそのまま開いて編集できます。画像は埋め込み済みなのでネット接続も `assets/` も不要。
- **フォント**：本ファイルは `Hiragino Sans`（Mac標準）で作成。Windowsでは自動で代替フォントに置換されます。社用PCで開いたら、全選択 → フォントを **`Meiryo` または `Yu Gothic`（游ゴシック）** に統一すると安定します。
- 文言・順序の微修正はここで完結します。

### B. スクリプトから再生成する（一括修正・差分管理向き）
講演の要点や構成をまとめて直したいときは `build_deck.js` を編集して作り直します。

```bash
# 1) Node.js（18以上）を用意
# 2) 依存をインストール
npm install -g pptxgenjs
# 3) 生成
cd <このフォルダ>
NODE_PATH=$(npm root -g) node build_deck.js
#   → ai_study_sdd.pptx が再生成される
```

画像で見た目を確認したい場合（LibreOffice が必要）：
```bash
soffice --headless --convert-to pdf "ai_study_sdd.pptx"
pdftoppm -jpeg -r 110 "ai_study_sdd.pdf" slide   # slide-01.jpg ... が出力される
```
> 社用PCにNode/LibreOfficeを入れられない場合は **Aの直接編集**で十分です。

## 4. 社用テンプレートへの載せ替え手順

本スライドは「載せ替え前提」で**装飾を最小限**にしてあります（左アクセントバー＋カード程度、全面カラーバーやアンダーライン装飾は不使用）。

おすすめの載せ替え方：
1. 社用テンプレートの新規ファイルを開く。
2. 本ファイルの各スライドを **テキスト・画像単位でコピー**し、テンプレートのプレースホルダに流し込む（スライドごと貼ると配色が混ざるため、要素単位推奨）。
3. 配色を社用カラーに置換（現行の対応 → 社用色を決めて一括置換）：
   - ネイビー `#1F3A5F`（見出し・濃色面）
   - ティール `#0E7C86`（アクセント・ラベル）
   - アンバー `#C2410C`（強調・「示唆」「結論」）
   - 濃色面の白文字／淡色面の濃文字、のコントラストは維持する。
4. フォントをテンプレート既定（Meiryo / Yu Gothic 等）に統一。
5. **公式ロゴ・登壇者写真**（`assets/`）は社内勉強会の引用として使用。差し替え・削除する場合は §7 を参照。

## 7. 画像・著作物の扱い（要確認）

- 表紙ロゴ・登壇者写真は**公開サミットサイトからの引用**。**社内限定の勉強会**での利用を前提としている。
- **社外公開・配布する場合**は要注意：登壇者の肖像／ロゴの再利用は権利者の許諾範囲を確認すること。社外資料化するなら写真は外す or 文字情報のみにするのが無難。
- 入れる場合は出典クレジット（例：最終ページに「写真・ロゴ：AI要件定義サミット2026公式サイトより」）を推奨。**現状は未記載**（壁打ち中の保留事項）。

## 8. デザイン仕様（テンプレ反映の参照値）

- レイアウト：16:9ワイド（13.33 × 7.5 inch）
- 配色：ネイビー `1F3A5F` / ティール `0E7C86` / アンバー `C2410C` / 本文 `23282D` / 補助 `6B7280` / 淡色面 `F2F5F7`

- 文字サイズ目安：スライドタイトル 23–28pt / セクション見出し 14–16pt / 本文 13–16pt / 注記 11–12pt
- モチーフ：見出し左の縦アクセントバー、角丸カード、「概要(紺)／気づき(ティール)／示唆(アンバー)」のラベルチップ
