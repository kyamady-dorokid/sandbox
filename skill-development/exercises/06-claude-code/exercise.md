# Claude Code 演習（MVP）

## 概要
マスタープラン「Claude Code基礎習得」に従った実践演習。
Claude Code のスラッシュコマンド、プロンプト工法、コード補完・生成機能を体験し、日常開発での活用方法を習得する。

## 学習ゴール
- Claude Code の基本操作（スラッシュコマンド、ファイル編集）ができる
- 効果的な自然言語プロンプトの書き方を習得する
- AI を使ったコード生成・リファクタリング・テスト生成を体験する
- 生成コードの品質を評価・改善できる

## 期待する成果物
- Claude Code を使って実装した簡易機能（例: バグ修正、小機能追加）
- AI 生成コードの品質評価レポート

## 典型的なワークフロー（最小例）
1. 対象ファイルを Claude Code で開く
2. スラッシュコマンド（例: `/edit`, `/create`, `/test`）で指示を出す
3. 生成されたコードをレビュー・修正
4. ローカルでテスト実行
5. 差分を確認して git に追加

### スラッシュコマンド例
- `/create`: 新規ファイル作成
- `/edit`: 既存ファイルを編集
- `/test`: テストを生成・実行
- `/refactor`: コードをリファクタリング

## 前提環境
- Claude Code が IDE/エディタで利用可能

## セットアップ手順（最小例）
1. IDE で Claude Code を有効化
2. テスト用の簡易プロジェクトを開く
3. スラッシュコマンドまたはプロンプトで指示を出す

## テスト要件
- 生成されたコードがローカルで動作すること
- 少なくとも 1 つのスラッシュコマンドを使用してコードが生成されること

## 完了条件
- [ ] Claude Code で 1 つ以上のファイルを正常に編集・作成できる
- [ ] 生成コードがローカルで動作する
- [ ] スラッシュコマンドの使い方を理解している

## 推定所要時間
- 仕様作成: 2〜4 時間（既に完了）
- 実践: 1〜3 時間

## 備考
- 複雑なプロンプトエンジニアリングは別で学習
- AI 生成コードの品質確保（セキュリティ、パフォーマンス）の重要性を理解すること
