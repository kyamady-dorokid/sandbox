## 概要
このIssueは「Claude Code基礎習得」項目の演習仕様を作成するタスクです。Claude Code の主要機能（スラッシュコマンド、コード補完、テスト支援など）を通じて日常開発での活用方法を示します。

## 学習ゴール
- Claude Code の基本的な操作（コマンド、ファイル編集、テスト実行）を理解する
- 自然言語プロンプトからの効果的な指示の出し方を学ぶ
- AI を用いたコード生成・リファクタリング・テスト生成の活用例を確認する

## 前提条件
- IDE/エディタ環境（もしくは Cloud 環境）で Claude Code が利用可能であること

## 成果物（Acceptance Criteria）
- `exercise.md` を作成し、このIssue に添付/リンクすること
- 実践ワークフロー（コマンド例、プロンプト例）と評価基準を提示すること

## スコープ（MVP）
- Claude Code の基本機能を使った 1〜2 個の典型的な開発フロー（例: バグ修正、簡易機能追加）を示す

## 作業タスク
1. `exercise.md` をテンプレートから作成
2. スラッシュコマンドやプロンプトのサンプルを作成
3. 評価基準（生成コードの品質チェック項目）を記載

## 推定作業時間
- 仕様作成: 2〜4 時間

## チェックリスト
- [ ] `exercise.md` を作成して添付/リンクする
- [ ] プロンプト/コマンド例を記載する
