## 概要
このIssueは「パブリッククラウドでのWEBサービス公開基礎」項目の演習仕様を作成するタスクです。AWS または GCP を使ったシンプルなアプリ公開手順（デプロイ、ストレージ、DB 接続、モニタリング）を想定します。

## 学習ゴール
- クラウドの基本サービス（Compute, Storage, Database）を用いたデプロイ手順を理解する
- IAM（認証・認可）やセキュリティの基本を理解する
- デプロイ後のモニタリングとログ確認の基本手順を経験する

## 前提条件
- 対象クラウドのアカウントが作成済みであること（学習環境での利用を想定）

## 成果物（Acceptance Criteria）
- `exercise.md` を作成し、このIssue に添付/リンクすること
- サンプルデプロイ手順（例: AWS Elastic Beanstalk / ECS / GCP App Engine / Cloud Run 等のいずれか）を明記すること

## スコープ（MVP）
- 最低限: シンプルな Web アプリをクラウドにデプロイし、外部からアクセス可能にする手順を完了できること

## 作業タスク
1. `exercise.md` をテンプレートから作成
2. AWS と GCP のどちらかを選定して具体的な手順を記載（選定は学習者の希望で可）
3. セキュリティ（IAM）・モニタリングの基本手順を追記

## 推定作業時間
- 仕様作成: 3〜6 時間

- [ ] デプロイ手順とモニタリング手順を記載する
